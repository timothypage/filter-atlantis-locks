# Atlantis Filter Locks

a super simple browser plugin to filter locks on the atlantis homepage, so you can see who's blocking you from applying and merging PRs! 


## installation

this is very much a vibe-coded WIP so use these instructions to run it "unpacked" https://developer.chrome.com/docs/extensions/get-started/tutorial/hello-world#load-unpacked
